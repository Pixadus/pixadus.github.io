Nikon D500 Cheat Sheet

Nov 20, 2019

Disclaimer: This sheet is being continuously updated.

Quick Reference:

Astrophotography:

  • EM: M or bulb
  • SS: 10” (higher allows for longer exposure, but obey the 500 rule.
  • f/stop: 1.8
  • ISO: 1600
  • Enable live view on camera, focus on brightest object possible. (Use highest possible shutter speed & ISO values here).

Exposure Mode:

  • P (Program): D500 selects basic exposure settings, but allows for user modification of all values while still maintaining metered exposure (different light values for different areas of image depending on relative brightness)
  • S (Shutter-priority): Useful to modify shutter speed to create either blur or exposure effects. f/stop is automatically chosen in this mode.
  • A (Aperture-priority): Choose to modify lens opening setting, to allow more or less light in when meter is “awake” (tap focus button i.e.)
  • M (Manual): Use this when you wish for full control over the shutter speed and f/stop. Manipulate shutter speed with main dial, and f/stop with sub-command dial.

Darktable:

Astrophotography:

  • RAW images allow the manipulation of the white balance to preferred values. Adjusting the temperature value to lower settings (~4000K) seems to have best effect.
